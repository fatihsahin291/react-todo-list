# To-Do List

This is a simple to-do list application built with React and Vite.

## Features

- Add new notes
- View note details
- Pin important notes

## Installation

First, clone the repository:

```sh
git clone <https://github.com/fatihsahin291/react-todo-list.git>
```

Then install the dependencies:

```sh
npm install
```

## Usage

To start the development server, run:

```sh
json-server --watch noteData.json
```

```sh
npm run start
```

To build the project for production, run:

```sh
npm run build
```
